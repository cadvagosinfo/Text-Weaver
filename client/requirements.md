## Packages
date-fns | Date formatting for the specific police report format
lucide-react | Icons for the UI

## Notes
The app requires a specific date format: DDHHMMMMMYY (e.g., 151630JAN25).
The cities list is dependent on the selected Unit ("41º BPM" vs "2ª Cia Ind").
The generated text report needs to be copyable to clipboard.
